# XRD Graph Maker v0.2

## 追加した主な機能
- 測定データを表示範囲内で `min → 0`, `max → 100` に規格化
- 参照ピークも**各相ごとに** `max = 100` へ自動規格化
- 複数の参照データを同時に読み込み可能
- 複数の参照相を**色分け**して表示可能
- 参照相ごとの表示名を自由に設定可能
- 測定データごとの表示名を自由に設定可能（空欄ならグラフ上では非表示）
- 配向フィルタを追加
  - `h00`
  - `0k0`
  - `00l`
  - `hk0`
  - `h0l`
  - `0kl`
- 複数の測定データを縦方向に積み上げて比較表示
- 下部の参照スティックを**複数参照相ごとに段分け**して表示
- x軸ラベル `2θ(deg.)` と `Cu-Kα` の重なりを修正

## 使い方
1. `xrd_graph_maker.html` をブラウザで開く
2. 測定データ `.xy` を読み込む
3. 参照データ `.txt` を読み込む
4. 「読み込み済みデータの表示名」で、測定データ名・参照相名を必要に応じて入力する
5. `2θ` 範囲、縦オフセット、配向フィルタ、最小参照強度などを調整する
6. `SVG保存` または `PNG保存` を行う

## 備考
- グラフ中のラベルは**ファイル名固定ではありません**。表示名欄に入力した文字が優先されます。
- 表示名が空欄なら、そのデータ名はグラフに表示しません。
- `show sticks` を有効にすると、複数参照相を下部で比較できます。
- `all` を選べば、複数参照相を同時に照合できます。
